### 使用 VUE UI 启动项目

```
vue ui
```

### 使用命令启动项目

```
// 编译和热重新加载以进行开发
npm run serve

// 为生产编译和压缩
npm run build

// 运行测试环境
npm run serve:test

// 运行测试

npm run test

// Lints and fixes files
npm run lint

// 运行单元测试
npm run test:unit
```
